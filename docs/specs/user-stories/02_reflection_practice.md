# Reflection & Practice User Stories

## Overview
AI-powered analysis, lenses, and practice tools.

### US-004: 성찰 Lens 선택
**As a** 사용자
**I want to** 대화를 분석할 관점(Lens)을 선택하고 싶다 (예: 공감, 논리, 갈등)
**So that** 다양한 측면에서 나의 대화 습관을 돌아볼 수 있도록.

- [x] AC-004-1: 분석 요청 시 '공감'(Default) 등 렌즈를 선택할 수 있다. (MVP: 공감 Lens 고정)

### US-005: AI 관찰 포인트 보기
**As a** 사용자
**I want to** AI가 분석한 주요 관찰 포인트와 성찰 질문을 보고 싶다
**So that** 객관적인 피드백을 통해 스스로를 성찰할 수 있도록.

- [x] AC-005-1: 대화 요약, 주요 포인트(3개), 성찰 질문이 표시된다.
- [x] AC-005-2: 분석 중일 때 로딩 상태를 확인할 수 있다.

### US-015: MITI 기반 정밀 평가 (Symmetric)
**As a** 전문적인 상담 수련생
**I want to** 두 화자 모두에 대해 상담자/내담자 관점의 정밀 평가를 받고 싶다
**So that** 역할극(Role Play)이나 상호 연습 상황에서 양쪽 모두 피드백을 얻을 수 있도록.

- [x] AC-015-1: '공감'(Empathy) 등 렌즈 선택 시 'MITI 정밀분석'을 선택할 수 있다.
- [x] AC-015-2: 분석 결과는 두 화자 각각에 대해 '상담 역량(Global/Behavior)'과 '내담자 반응(Change Talk)'을 모두 보여준다.
- [x] AC-015-3: 점수는 차트(Radar Chart 등)로 시각화되어 직관적으로 비교할 수 있다.

### US-006: 연습 카드 (Practice Cards)
**As a** 사용자
**I want to** 내가 놓친 결정적 순간들에 대해 "더 나은 대화"를 제안받고 싶다
**So that** 단순히 점수만 보는 게 아니라 구체적인 개선 방법을 학습할 수 있도록.

- [x] AC-006-1: 분석 결과 하단에 '연습 카드' 섹션이 제공된다 (세션 당 3개 내외).
- [x] AC-006-2: 카드는 [상황] -> [내가 한 말] -> [MITI 진단] -> [모범 답안] 순서로 구성된다.
- [x] AC-006-3: 카드를 뒤집는 인터랙션을 통해 정답을 확인하는 학습 경험을 제공한다.

### US-007: 연습 카드 저장
**As a** 사용자
**I want to** 제안받은 연습 카드 중 유용한 것을 보관함에 저장하고 싶다
**So that** 나중에 다시 꺼내보며 반복 학습(Drill)을 할 수 있도록.

- [ ] AC-007-1: 연습 카드에 '저장(Save)' 또는 '북마크' 버튼이 있다.
- [ ] AC-007-2: 저장된 카드는 별도의 '내 연습(My Practice)' 메뉴에서 모아볼 수 있다.
- [ ] AC-007-3: 저장된 카드를 삭제(저장 취소)할 수 있다.
