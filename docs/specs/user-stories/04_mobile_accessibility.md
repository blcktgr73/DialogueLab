# Mobile & Accessibility User Stories

## Overview
User stories related to mobile web experience and progression to installed apps (PWA).

### US-020: 모바일 접근성 (PWA)
**As a** 모바일 사용자
**I want to** 앱스토어 설치 없이도 이 서비스를 앱처럼 폰에 설치해서 쓰고 싶다
**So that** 매번 브라우저를 켜지 않고 홈 화면에서 바로 접근하여 연습할 수 있도록.

- [ ] AC-020-1: `manifest.json`이 설정되어 "홈 화면에 추가(Add to Home Screen)"가 가능하다.
- [ ] AC-020-2: 모바일 브라우저에서도 UI가 깨지지 않고 반응형(Responsive)으로 잘 보인다. (특히 녹음 인터페이스)
- [ ] AC-020-3: 오프라인 상태에서도 기본적인 앱 셸(Shell)이 로드되어야 한다.
