# User Story Governance & Stewardship

## ID Naming Conventions
- **Themes**: `THEME-XX` (Large functional areas)
- **Epics**: `EPIC-XX` (Major features)
- **User Stories**: `US-XXX` (Individual implementable units)

## Stewardship Policy
1. **Definition**: All stories must follow the template "As a... I want to... So that...".
2. **Acceptance Criteria**: Must be measurable and testable.
