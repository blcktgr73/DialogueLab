# Session Management & Data User Stories

## Overview
Core functionality for creating, managing, and storing conversation sessions.

### US-001: 세션 생성
**As a** 사용자 (User)
**I want to** 새로운 대화 연습 세션을 시작하고 싶다
**So that** 성찰 과정을 시작할 수 있도록.

- [x] AC-001-1: 사용자는 제목을 입력하여 세션을 생성할 수 있다.
- [x] AC-001-2: 사용자는 "대화 모드"를 선택할 수 있다 (자유 / 연습 / 평가).

### US-014: 세션 목록 및 대시보드
**As a** 사용자
**I want to** 내가 진행했던 과거 세션들의 목록을 보고 싶다
**So that** 나의 연습 기록을 확인하고 다시 들어가서 공부할 수 있도록.

- [ ] AC-014-1: 홈 화면에서 날짜순으로 정렬된 내 세션 목록을 볼 수 있다.
- [x] AC-014-2: 세션 클릭 시 상세 페이지로 이동한다.

### US-016: 세션 관리 (수정/삭제)
**As a** 사용자
**I want to** 세션의 이름을 바꾸거나 불필요한 세션을 삭제하고 싶다
**So that** 내 대화 라이브러리를 깔끔하게 정리할 수 있도록.

- [x] AC-016-1: 대시보드 또는 상세 화면에서 세션 이름을 수정할 수 있다.
- [x] AC-016-2: 대시보드에서 세션을 영구 삭제할 수 있다.
- [x] AC-016-3: 파일 업로드 시, 기본 이름인 경우 파일명으로 자동 변경된다.

### US-017: 화자 이름 변경 (Bulk Update)
**As a** 사용자
**I want to** 대화 내용의 화자(Speaker) 이름을 수정하고 싶다
**So that** '화자1', '참여자A' 같이 불명확한 이름을 실제 이름으로 정리할 수 있도록.

- [x] AC-017-1: 대화 목록에서 화자 이름을 클릭하여 수정할 수 있다.
- [x] AC-017-2: 화자 이름을 수정하면, 해당 세션 내의 동일한 화자는 모두 일괄 변경된다 (Bulk Update).

### US-013: 축어록 파일 업로드
**As a** 사용자
**I want to** 기존에 가지고 있는 축어록 파일(TXT, Excel 등)을 업로드하고 싶다
**So that** 일일이 타이핑하지 않고 대화 내용을 시스템에 등록할 수 있도록.

- [x] AC-013-1: 텍스트 파일(.txt)을 파싱하여 대화 내용으로 변환한다. (Format: "화자: 내용")
- [x] AC-013-2: 엑셀/CSV 파일(.csv, .xls, .xlsx)을 파싱하여 대화 내용으로 변환한다. (Columns: Speaker, Content, Timestamp)

### US-019: 다자간 음성 인식 (STT)
**As a** 사용자
**I want to** 다수의 화자가 참여한 음성 파일을 업로드하여 자동으로 텍스트로 변환하고 싶다 (Google Cloud STT)
**So that** 일일이 타이핑하거나 화자를 수동으로 구분하는 수고를 덜 수 있도록.

- [ ] AC-019-1: 음성 파일(MP3, WAV 등)을 업로드하면 Google Cloud STT를 통해 텍스트로 변환된다.
- [ ] AC-019-2: Speaker Diarization(화자 분리) 기능을 통해 화자(Speaker 1, Speaker 2...)가 자동으로 구분된다.
- [ ] AC-019-3: 최대 10명 이상의 화자를 구분할 수 있어야 한다.
- [ ] AC-019-4: 변환 완료 후, 사용자는 내용을 검토하고 수정할 수 있다.
