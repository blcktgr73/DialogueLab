# Identity & Collaboration User Stories

## Overview
User accounts, profiles, and collaborative features.

### US-008: Google 로그인
**As a** 사용자
**I want to** Google 계정으로 로그인하고 싶다
**So that** 내 세션 기록을 저장하고 다른 사람을 초대할 수 있도록.

- [x] AC-008-1: Google OAuth 로그인을 지원한다.
- [x] AC-008-2: 첫 로그인 시 프로필(이름) 정보를 가져와 저장한다.

### US-009: 사용자 프로필 설정
**As a** 사용자
**I want to** 나의 표시 이름(닉네임)을 변경하고 로그아웃 할 수 있다.
**So that** 익명성을 유지하거나 본계정을 안전하게 관리할 수 있도록.

- [x] AC-009-1: 프로필 페이지에서 닉네임을 수정하고 저장할 수 있다.
- [x] AC-009-2: 로그아웃을 하면 로그인 페이지로 이동한다.

### US-010: 사용자 검색 및 초대
**As a** 사용자
**I want to** 이름을 검색하여 파트너를 초대하고 싶다
**So that** 내 대화 기록을 공유하고 함께 분석할 수 있도록.

- [x] AC-010-1: 이메일 또는 이름(2글자 이상)으로 사용자를 검색할 수 있다.
- [x] AC-010-2: 검색 결과에는 이름, 아바타, 마스킹된 이메일이 표시된다.
- [x] AC-010-3: '초대' 버튼을 누르면 해당 사용자에게 세션 접근 권한이 부여된다.

### US-011: 참여자 관리
**As a** 세션 소유자 (Owner)
**I want to** 내 세션에 누가 들어와 있는지 보고, 내보낼 수 싶다
**So that** 세션의 보안을 관리할 수 있도록.

- [x] AC-011-1: 세션 상세 화면에서 현재 참여자 목록을 볼 수 있다.
- [x] AC-011-2: 참여자를 목록에서 제거(권한 해제)할 수 있다.
