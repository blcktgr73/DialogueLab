# User Stories: Agentic Simulation (MI Training)

## Overview
User stories related to the real-time Standardized Patient simulation for Motivational Interviewing (MI) training.

| ID | Story | Priority | Status |
| :--- | :--- | :--- | :--- |
| **US-021** | **시뮬레이션: 파트너 페르소나 설정**<br>사용자는 훈련하고 싶은 파트너의 이름, 상담 주제(금연 등), 저항 수준을 설정하여 맞춤형 연습을 준비할 수 있다. | High | ⏳ |
| **US-022** | **시뮬레이션: 실시간 음성 대화 (Gemini Live)**<br>사용자는 설정된 AI 파트너와 지연 없는 실시간 음성 대화를 할 수 있으며, 중간에 말을 끊거나(Barge-in) 감정을 실어 대화할 수 있다. | High | ⏳ |
| **US-023** | **시뮬레이션: 자동 종료 및 모드 전환**<br>대화가 자연스럽게 마무리되거나 제한 시간(15분)이 되면, 시스템은 자동으로 시뮬레이션을 종료하고 분석 모드로 전환한다. | Medium | ⏳ |
| **US-024** | **시뮬레이션: 멀티모달 피드백**<br>사용자는 대화 내용뿐만 아니라 목소리 톤(권위적, 따뜻함 등)이 파트너에게 미친 영향에 대해 피드백을 받을 수 있다. | Medium | ⏳ |
| **US-025** | **시뮬레이션: MI 기술 분석 리포트**<br>사용자는 세션 종료 후 OARS 사용 빈도, 변화 대화(Change Talk) 포착 능력 등 MI 원칙에 기반한 상세 분석 결과를 확인할 수 있다. | High | ⏳ |
