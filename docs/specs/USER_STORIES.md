# 사용자 스토리 및 인수 기준 (User Stories & Acceptance Criteria)

## MVP (Alpha)

### US-001: 세션 생성
**As a** 사용자 (User)
**I want to** 새로운 대화 연습 세션을 시작하고 싶다
**So that** 성찰 과정을 시작할 수 있도록.

- [x] AC-001-1: 사용자는 제목을 입력하여 세션을 생성할 수 있다.
- [x] AC-001-2: 사용자는 "대화 모드"를 선택할 수 있다 (자유 / 연습 / 평가).

### US-014: 세션 목록 및 대시보드
**As a** 사용자
**I want to** 내가 진행했던 과거 세션들의 목록을 보고 싶다
**So that** 나의 연습 기록을 확인하고 다시 들어가서 공부할 수 있도록.

- [ ] AC-014-1: 홈 화면에서 날짜순으로 정렬된 내 세션 목록을 볼 수 있다.
- [x] AC-014-2: 세션 클릭 시 상세 페이지로 이동한다.

### US-016: 세션 관리 (수정/삭제)
**As a** 사용자
**I want to** 세션의 이름을 바꾸거나 불필요한 세션을 삭제하고 싶다
**So that** 내 대화 라이브러리를 깔끔하게 정리할 수 있도록.

- [x] AC-016-1: 대시보드 또는 상세 화면에서 세션 이름을 수정할 수 있다.
- [x] AC-016-2: 대시보드에서 세션을 영구 삭제할 수 있다.
- [x] AC-016-3: 파일 업로드 시, 기본 이름인 경우 파일명으로 자동 변경된다.

### US-017: 화자 이름 변경 (Bulk Update)
**As a** 사용자
**I want to** 대화 내용의 화자(Speaker) 이름을 수정하고 싶다
**So that** '화자1', '참여자A' 같이 불명확한 이름을 실제 이름으로 정리할 수 있도록.

- [ ] AC-017-1: 대화 목록에서 화자 이름을 클릭하여 수정할 수 있다.
- [ ] AC-017-2: 화자 이름을 수정하면, 해당 세션 내의 동일한 화자는 모두 일괄 변경된다 (Bulk Update).

### US-008: Google 로그인
**As a** 사용자
**I want to** Google 계정으로 로그인하고 싶다
**So that** 내 세션 기록을 저장하고 다른 사람을 초대할 수 있도록.

- [x] AC-008-1: Google OAuth 로그인을 지원한다.
- [x] AC-008-2: 첫 로그인 시 프로필(이름) 정보를 가져와 저장한다.

### US-009: 사용자 프로필 설정
**As a** 사용자
**I want to** 나의 표시 이름(닉네임)을 변경하고 로그아웃 할 수 있다.
**So that** 익명성을 유지하거나 본계정을 안전하게 관리할 수 있도록.

- [x] AC-009-1: 프로필 페이지에서 닉네임을 수정하고 저장할 수 있다.
- [x] AC-009-2: 로그아웃을 하면 로그인 페이지로 이동한다.

### US-010: 사용자 검색 및 초대
**As a** 사용자
**I want to** 이름을 검색하여 파트너를 초대하고 싶다
**So that** 내 대화 기록을 공유하고 함께 분석할 수 있도록.

- [x] AC-010-1: 이메일 또는 이름(2글자 이상)으로 사용자를 검색할 수 있다.
- [x] AC-010-2: 검색 결과에는 이름, 아바타, 마스킹된 이메일이 표시된다.
- [x] AC-010-3: '초대' 버튼을 누르면 해당 사용자에게 세션 접근 권한이 부여된다.

### US-011: 참여자 관리
**As a** 세션 소유자 (Owner)
**I want to** 내 세션에 누가 들어와 있는지 보고, 내보낼 수 싶다
**So that** 세션의 보안을 관리할 수 있도록.

- [x] AC-011-1: 세션 상세 화면에서 현재 참여자 목록을 볼 수 있다.
- [x] AC-011-2: 참여자를 목록에서 제거(권한 해제)할 수 있다.

### US-013: 축어록 파일 업로드
**As a** 사용자
**I want to** 기존에 가지고 있는 축어록 파일(TXT, Excel 등)을 업로드하고 싶다
**So that** 일일이 타이핑하지 않고 대화 내용을 시스템에 등록할 수 있도록.

- [x] AC-013-1: 텍스트 파일(.txt)을 파싱하여 대화 내용으로 변환한다. (Format: "화자: 내용")
- [x] AC-013-2: 엑셀/CSV 파일(.csv, .xls, .xlsx)을 파싱하여 대화 내용으로 변환한다. (Columns: Speaker, Content, Timestamp)
---

### US-004: 성찰 Lens 선택
**As a** 사용자
**I want to** 대화를 분석할 관점(Lens)을 선택하고 싶다 (예: 공감, 논리, 갈등)
**So that** 다양한 측면에서 나의 대화 습관을 돌아볼 수 있도록.

- [x] AC-004-1: 분석 요청 시 '공감'(Default) 등 렌즈를 선택할 수 있다. (MVP: 공감 Lens 고정)

### US-005: AI 관찰 포인트 보기
**As a** 사용자
**I want to** AI가 분석한 주요 관찰 포인트와 성찰 질문을 보고 싶다
**So that** 객관적인 피드백을 통해 스스로를 성찰할 수 있도록.

- [x] AC-005-1: 대화 요약, 주요 포인트(3개), 성찰 질문이 표시된다.
- [x] AC-005-2: 분석 중일 때 로딩 상태를 확인할 수 있다.

### US-015: MITI 기반 정밀 평가 (Symmetric)
**As a** 전문적인 상담 수련생
**I want to** 두 화자 모두에 대해 상담자/내담자 관점의 정밀 평가를 받고 싶다
**So that** 역할극(Role Play)이나 상호 연습 상황에서 양쪽 모두 피드백을 얻을 수 있도록.

- [x] AC-015-1: '공감'(Empathy) 등 렌즈 선택 시 'MITI 정밀분석'을 선택할 수 있다.
- [x] AC-015-2: 분석 결과는 두 화자 각각에 대해 '상담 역량(Global/Behavior)'과 '내담자 반응(Change Talk)'을 모두 보여준다.
- [x] AC-015-3: 점수는 차트(Radar Chart 등)로 시각화되어 직관적으로 비교할 수 있다.

### US-018: 연습 카드 (Practice Cards)
**As a** 사용자
**I want to** 내가 놓친 결정적 순간들에 대해 "더 나은 대화"를 제안받고 싶다
**So that** 단순히 점수만 보는 게 아니라 구체적인 개선 방법을 학습할 수 있도록.

- [x] AC-018-1: 분석 결과 하단에 '연습 카드' 섹션이 제공된다 (세션 당 3개 내외).
- [x] AC-018-2: 카드는 [상황] -> [내가 한 말] -> [MITI 진단] -> [모범 답안] 순서로 구성된다.
- [x] AC-018-3: 카드를 뒤집는 인터랙션을 통해 정답을 확인하는 학습 경험을 제공한다.
