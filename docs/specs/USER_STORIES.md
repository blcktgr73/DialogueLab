# 사용자 스토리 및 인수 기준 (User Stories & Acceptance Criteria)

## MVP (Alpha)

### US-001: 세션 생성
**As a** 사용자 (User)
**I want to** 새로운 대화 연습 세션을 시작하고 싶다
**So that** 성찰 과정을 시작할 수 있도록.

- [x] AC-001-1: 사용자는 제목을 입력하여 세션을 생성할 수 있다.
- [x] AC-001-2: 사용자는 "대화 모드"를 선택할 수 있다 (자유 / 연습 / 평가).

### US-014: 세션 목록 및 대시보드
**As a** 사용자
**I want to** 내가 진행했던 과거 세션들의 목록을 보고 싶다
**So that** 나의 연습 기록을 확인하고 다시 들어가서 공부할 수 있도록.

- [ ] AC-014-1: 홈 화면에서 날짜순으로 정렬된 내 세션 목록을 볼 수 있다.
- [ ] AC-014-2: 세션 클릭 시 상세 페이지로 이동한다.

### US-008: Google 로그인
**As a** 사용자
**I want to** Google 계정으로 로그인하고 싶다
**So that** 내 세션 기록을 저장하고 다른 사람을 초대할 수 있도록.

- [x] AC-008-1: Google OAuth 로그인을 지원한다.
- [x] AC-008-2: 첫 로그인 시 프로필(이름) 정보를 가져와 저장한다.

### US-009: 사용자 프로필 설정
**As a** 사용자
**I want to** 나의 표시 이름(닉네임)을 변경하고 로그아웃 할 수 있다.
**So that** 익명성을 유지하거나 본계정을 안전하게 관리할 수 있도록.

- [x] AC-009-1: 프로필 페이지에서 닉네임을 수정하고 저장할 수 있다.
- [x] AC-009-2: 로그아웃을 하면 로그인 페이지로 이동한다.

### US-010: 사용자 검색 및 초대
**As a** 사용자
**I want to** 이름을 검색하여 파트너를 초대하고 싶다
**So that** 내 대화 기록을 공유하고 함께 분석할 수 있도록.

- [ ] AC-010-1: 이메일 또는 이름으로 사용자를 검색할 수 있다.
- [ ] AC-010-2: 검색된 사용자에게 초대(공유 권한)를 보낼 수 있다.

- [ ] AC-010-2: 검색된 사용자에게 초대(공유 권한)를 보낼 수 있다.

### US-013: 축어록 파일 업로드
**As a** 사용자
**I want to** 기존에 가지고 있는 축어록 파일(TXT, Excel 등)을 업로드하고 싶다
**So that** 일일이 타이핑하지 않고 대화 내용을 시스템에 등록할 수 있도록.

- [x] AC-013-1: 텍스트 파일(.txt)을 파싱하여 대화 내용으로 변환한다. (Format: "화자: 내용")
- [x] AC-013-2: 엑셀/CSV 파일(.csv, .xls, .xlsx)을 파싱하여 대화 내용으로 변환한다. (Columns: Speaker, Content, Timestamp)
---

### US-004: 성찰 Lens 선택
**As a** 사용자
**I want to** 대화를 분석할 관점(Lens)을 선택하고 싶다 (예: 공감, 논리, 갈등)
**So that** 다양한 측면에서 나의 대화 습관을 돌아볼 수 있도록.

- [x] AC-004-1: 분석 요청 시 '공감'(Default) 등 렌즈를 선택할 수 있다. (MVP: 공감 Lens 고정)

### US-005: AI 관찰 포인트 보기
**As a** 사용자
**I want to** AI가 분석한 주요 관찰 포인트와 성찰 질문을 보고 싶다
**So that** 객관적인 피드백을 통해 스스로를 성찰할 수 있도록.

- [x] AC-005-1: 대화 요약, 주요 포인트(3개), 성찰 질문이 표시된다.
- [x] AC-005-2: 분석 중일 때 로딩 상태를 확인할 수 있다.

### US-015: MITI 기반 정밀 평가
**As a** 전문적인 상담 수련생
**I want to** 동기면담(MITI) 코드에 기반한 정밀한 평가를 받고 싶다
**So that** 단순 조언이 아닌 전문적인 상담 기술 점수를 확인하고 개선하기 위해.

- [ ] AC-015-1: AI가 대화 내용을 MITI 코드(예: 공감, 지지, 자율성 존중 등)로 분류한다.
- [ ] AC-015-2: 전반적인 정신(Global Spirit) 점수와 행동 코드 빈도를 보여준다.
